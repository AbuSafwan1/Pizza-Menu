# 🍕 Fast React Pizza Co.

A professional pizza restaurant menu built with React.

## Tech Stack
- React 18
- CSS3 (Custom Properties, Grid, Flexbox)
- Google Fonts (Cormorant Garamond + DM Sans)

## Getting Started

### Install dependencies
npm install

### Run development server
npm start

### Build for production
npm run build

## Features
- Dynamic menu with sold-out detection
- Real-time open/closed status based on current hour
- Reservation modal
- Fully responsive design
